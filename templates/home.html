The QueryCARE function is accessible to any user. QueryCARE allows users to search information about motifs in the database.

Any user can access the SeachForCARE function. SeachForCARE allows users to search within the database.

Any user can access the MotifSampler function. MotifSampler is a function that discovers new motifs based on the MotifSampler algorithm..

Only users who have logged in to the system have the function to review history and of course search history is only saved when you are logged in.

The Biologist function is a special function for biologists. This functionality allows biologists to validate and populate motifs generated from the MotifSampler tool. Since then, the database has become increasingly rich and highly accurate.
